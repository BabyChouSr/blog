A Dictionary is a Mapping

A dictionary is like a list, but it contains both keys and values. A dictionary is essentially a collection of indices that contain this key-value pair where each index also has a value associated with it.

In math, this is known as a mapping from keys to values since each links or "maps" to a certain value.

Let's create a dictionary that translates numbers to english.

We start with an empty dictionary that looks like this:

numToEng = dict()

Notice the squiggly-brackets.

We create a certain key to value mapping by doing

numToEng[1] = 'one'

This creates a pair of {1 : 'one'}

We can do this for several keys and values:

numToEng[1] = 'one'
numToEng[2] = 'two'
numToEng[3] = 'three'
numToEng[4] = 'four'

Our dictionary would then look like:

{1 : 'one', 2: 'two', 3 : 'three', 4: 'four'}

We access any value by using the index which in this case is the key.

numToEng[1] = 'one'

1 is the key and 'one' is the value

Traversing Dictionaries

If you use a dictionary in a for statement, it traverses the keys of the dictionary:

numToEng = {1 : 'one', 2: 'two', 3 : 'three', 4: 'four'}
for i in numToEng:
	print(numToEng[i])
>>> one
>>> two
>>> three
>>> four

i represents each key within numToEng which we traverse to access each value.

Reverse Lookup

It is very easy to find for some key k a certain value v since we just have to use a for loop until we find the key that corresponds to k.

However, let's say we wanted to use the value v to find the corresponding value k. We would have to do something called reverse lookup.

For example, let's say we want to look up for the number that corresponds to 'four'

numToEng = {1 : 'one', 2: 'two', 3 : 'three', 4: 'four'}
def reverse_lookup(dict, value):
	for i in dict:
		if dict[i] == value:
			return i
	return None
print(reverse_lookup(numToEng, 'four'))
>>> 4

Running this code, we can insert a certain value v and would be able to find the key that corresponds to that value.
