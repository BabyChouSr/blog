Values and Types

A value is a basic thing such as numbers or letters. A value can be something like 3, 32.0, or "Hey." These values are of different types.

  • 3 is an integer because it cannot have decimals after it
  • 32.0 is a floating-point number because it is has decimals afterwards. Integers are a subset of floating-point numbers since floating-point numbers include all numbers that don't have decimals afterwards and those that do.
  • "Hey" is a string because it contains characters or letters instead of numbers

Assignment Statements

As assignment statement gives a value to a certain variable.

This can look something like this:

x = 8
y = 32.234234
message = "Hi, my name is Chris"

Variable Names

By convention, programmers use something known as camelCase where the first word is lowercase and every word afterwards would be Capitalized. We also often separate the words in variables using underscores.

Good Name Examples:

  • nameOfSister
  • myBrother
  • set_of_oranges
  • my_thing_of_beauty

Note that we cannot start our variable name with numbers. Also, note there are some words that we can't use because they are python keywords reserved to call certain functions or reference pre-defined structures or variables.

These include:

None       continue   for        lambda     try
True       def        from       nonlocal   while
and        del        global     not        with
as         elif       if         or         yield
assert     else       import     pass
break      except     in         raise

Expressions and Statements

Expressions are a combination of values, variables, and operators.

This could be something like:

  • 17
  • n+2
  • n/10

Statements create a variable or display a value:

  • n = 5
  • print("Hello World')

Arithmetic Operators and Order

All expressions follow the order known as PEMDAS which tell us the order of operations is

  • Parantheses
  • Exponents
  • Multiplication
  • Division
  • Addition
  • Subtraction

We are allowed to do various arithmetic operations such as:

x = 2
n = 4
print(x+n) #Outputs 6

n = x + 5 # n is now 7

String Operations

When we want to combine string, we can do a procedure known as concatenation

Therefore, adding Strings would look something like:

x = "Hey"
y = " boy"
z = x + y
print(z) # Outputs Hey boy

We can even multiply Strings. This will cause the original string to be duplicated various times

x = "woah"
print(x *3)#Outputs woahwoahwoah

Comments
