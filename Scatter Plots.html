Importing Libraries

As always, we have to import the libraries we are using

import numpy as np
import matplotlib.pyplot as plt

%matplotlib inline

Getting Started

I will be utilizing random data to show you guys how to get started. x and y are just random arrays from 0 to 10.

Notice how I now use plt.scatter() which accepts the x-coordinates, y-coordinates, the size of the dots, the color, and the label of the plot. 

x =  np.random.rand(10) * 10
y = np.random.rand(10) * 10
plt.style.use('bmh')
plt.scatter(x,y,s = 100, c= '#32a879', label = "Randomness")
plt.xlabel('x')
plt.ylabel('y')
plt.title("Scattered")
plt.legend()
plt.show()

If we simply tweak the size of the dots to something like 300, we get something that looks likes this:

Satisfaction Rating

Imagine if we were working at retail and we wanted to show the satisfaction rating of each person. Now, we would have to have each dot a different color/gradient based on their satisfaction. Notice, this works with anything that relies on a scale, it doesn't have to be satisfaction rating.

There are several things that I added that I will elaborate on:

  1. The color array are numbers demonstrating the gradient from 0 to 10
  1. The scatter now accepts "c" as its color or gradient array. We have to input a different color mapping in order for there to be a gradient. In this case, I put the color map of Blues.
  1. I created a colorbar and the set the label as satisfaction rating to provide a label for the data essentially.
color = np.random.rand(10) * 10
plt.scatter(x,y,s = 300, c= color,cmap = 'Blues',label = "Randomness")
cbar = plt.colorbar()
cbar.set_label('Satisfaction Rating')

With that in mind, we get a plot that looks something like this:

Playing Around

I decided to just play around with a couple of the sizes utilizing an array.

My new code looks something like this

x =  np.random.rand(100) * 100
y = np.random.rand(100) * 1000
color = np.random.rand(100) * 10
size = np.random.rand(100) * 300 + 50
plt.style.use('bmh')
plt.scatter(x,y,s = size, c= color,cmap = "viridis",label = "Randomness")
plt.xlabel('x')
plt.ylabel('y')
plt.title("Scattered")
plt.legend()
plt.show()

I now have 100 elements all with different sizes that I pass in using the size array that was generated randomly.
