String as a Sequence

A string is a sequence of characters. You can access the characters one at a time. 

word = "apple"
letter = word[0]

The expression in the brackets is called the index. It indicates which character we want.

Thus, if we were to print out the string letter, we get 'a'

The first letter does not start with index 1 like most people might think, but instead 0.

a p p l e
0 1 2 3 4

The Length of a String

The built in len feature returns the number of characters in a string.

person = "Charlie"
len(person)
>>> 7

The length prints out the number of characters in a string.

Substrings

A segment of a string is called substring.

To see this in action

s = "Christopher Chou"
firstName = s[0:11]
>>> 'Christopher'
lastName = s[12:16]
>>> 'Chou'

The operator [n:m] returns the part of the string from the "n-eth" character to the "m-eth" character, including the first but excluding the last.

We could do a shortcut to traverse from a certain index to the very end by just omitting the second index.

fruit = 'banana'
>>> fruit[3:]
'ana'
>>> fruit[0:]
'banana'

Strings are immutable

We cannot change an existing String.

Something like 

message = "hey bro"
message[0] = "b"

Is not allowed because item assignment is not supported.

Conditional Statements

Before going through conditional statements, we must first define a new type called the boolean

The Boolean evaluates whether a statement is True or False

We could use this to check if certain statements are true and then do an operation with that.

We create something called if-elif-else statements which allows us to test through various cases.

Example of the Structure

if <condition 1>:

do something

elif <condition 2>:

do something:

else :

do something

What happens is we check the first condition and if that is true then we do the action and don't check after that.

If we fail condition one, we check for condition 2 then if that is true, we do the action and jump out.

Lastly, we do the action in the else statement if it failed all conditions.

Operators

  1. ==: Checks if two quantities are equal
  1. > : Checks if one quantity is greater than the other
  1. <: Checks if one quantity is less than the other
  1. !=: Checks to see if two quantities are not equal.

Case: String Comparison

We can compare the lengths of strings by using these conditional statements.

In plain English, let's say we wanted to compare two words: 'apple' and 'orange' to see if they are equal. Obviously, they are not since they are different words.

word_1 = 'apple'
word_2 = 'orange'
if word_1 == word_2:
	print('True')
else:
	print('False')

>>> 'False'

This statement prints out a False since it is obvious that the words are not equal. We check equality by using ==

Loops

Traversing a string with a for loop

A for loops works like this: we loop over every letter in a certain String.

Putting this in code

s = "Chris"
for i in s:
	print(i)
>>> C
>>> h
>>> r
>>> i 
>>> s

We traverse through every letter just like that.

We could rewrite this as a while loop.

This would look something like this.

s = 'Chris'
index = 0
while index < len(s):
	print(s[index])
	index = index + 1
>>> C
>>> h
>>> r
>>> i
>>> s

We start with the 0th index which corresponds with the first letter and iterate all the way until the last index which is len(s) - 1 We have to make sure to update the index with each iteration or this would be an infinite loop.
