Intro to Convolutional Neural Networks

Convolutional Neural Networks (CNNs) are incredibly useful in various tasks but in this blog, I will go over the process for image classification.

Convolutional Layer

The role of the convolutional layer is to reduce the images into a form which is easier to process without losing the important features that could help with predictions.

We utilize something called filters or convolutional kernels in order to highlight what we are going to be detecting. In the example below, we see several filters that are used to help detect features. Different features will have different weights that will help with detecting the features.

Filters are applied by starting at the top left of the input, and going to the right with a certain stride amount and then going down. The kernel shifts to the right while applying matrix multiplication over the area that it goes over.

For example, let's say it goes over the example above in the top left and we are applying a vertical line filter, mathematically, we would get

We would then shift this same layer to the right by a certain stride amount and do it again.

With each filter, we are able to detect certain small features of the image. There is a hierarchy however. As we go from the first convolution layer to the next, the features get more and more complicated until we start detecting actual objects

As we can see from this image, we start at just lines, then certain features on the face, until we reach a person's face.

We could also add something called padding which allows us to put zeroes on the boundaries of the image. This allows to have a more accurate analysis of the image.

Pooling Layer

The main objective of the pooling layer is to reduce the input image in order to reduce the computational load on the machine.

It's just like the filter and has a size, stride, and padding type but it doesn't have any waits and it just aggregates the data. For example, a max pooling layer makes it so only the maximum value in the field moves on or a mean pooling layer makes it so only the average value moves on.

It helps with de-noising the data as well as reducing the dimensions of the data, making computational speeds faster and more accurate.

Fully Connected Layer

This layer helps with the classification of the image. For example, if we are trying to predict a cat, it helps to tell the network the features most correlated with a cat like claws or whiskers. If we were predicting a dog, we'd expect a tail or paws. This layer helps with the classification of the image as it is fed into a feed-forward neural network like a SoftMax unit.
