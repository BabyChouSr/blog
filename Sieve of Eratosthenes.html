Sieve of Eratosthenes

The concept behind the Sieve of Eratosthenes in Number Theory is that any number that is not a prime number can be derived from prime numbers. Thus, to find a prime number, we only have to find the numbers that are not divisible from previously found prime numbers.

This shouldn't come as a surprise to anyone that has learned prime factorization. After all, we are taught that any number can be simplified into powers of prime numbers. For example, 8=238 = 2^38=23﻿ or 75=52⋅375 = 5^2 \cdot 375=52⋅3﻿ where 2,3,5 are all prime numbers.

Code in C++

#include <iostream>
#include <vector>
using namespace std;

bool Prime(int n, vector<int> p) {
    for (int i = 0; i < p.size(); i++){
        if (n %  p[i] == 0)  {return false;}
    }
    return true;
}
int main(){
    int n = 100;
    vector<int> primes;
    for (int i = 2; i <= n; i++){
        if (Prime(i,primes)) {
            primes.push_back(i);
        }
    }
}

This code allows to check for each prime from 2 to 100.

We create a vector of all the prime numbers and check using the Prime function if the number is divisible or not by each of the numbers in the vector of primes. If the number is divisible shown in if (n % p[i] == 0) then we return false because the number is not false. Otherwise, the number is a prime since it is not divisible by any prime number. We do this step for every number from 2 to 100 as shown by the for loop.
