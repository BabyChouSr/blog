Logistic Regression

Logistic Regression is incredibly important in Data Science and Machine Learning because it gets an input x and outputs either a 1 or 0. This is useful because it allows us to do binary classification which is incredibly powerful in application.

Types of Logistic Regression

  1. Binary (The Regular 1s and 0s)
  1. Multi-Class (Any amount of numbers: 0,1,2....)
