Function Calls

An example of a function call is 

>>> type(16)
<type 'int'>

The function name is type

The expression in parentheses is the argument

The result of the function is called the return value

Type Conversion Functions

Python has several built-in functions that convert values from one type to another.

The int function takes any value and converts it to an integer.

The float function converts integers and strings to floating-point numbers.

The str function converts its argument to a string

>>> int('32')
32
>>> int(3.23412)
3
>>> float(32)
32.0
>>> str(3.123213)
'3.123213'

Math Functions

Python has several mathematical functions that can be accessed through the module math.

We first import it by stating

import math

To utilize the functions within math, we specify the name of the module and the name of the function separated by a dot. This is known as dot notation.

# Examples
x = math.sin(123)
y = math.cos(132)
z = math.sqrt(123)

You can get some constants such as pi by using dot notation as well

math.pi

Composition

We can put entire expressions and even other functions within functions.

This would look something like this

x = math.sin(123 * 360 / 2465)

Notice the use of arithmetic operators yet it still works.

We can also put other functions:

x = math.exp(math.log(x+1))

We utilize the log function within the exponential function and it works.

Function Definition

Example:

def boy():
	print("I am indeed a boy")

def is a keyword that indicates function definition.

The name of the function is boy

The empty parentheses indicates that this function doesn't take any arguments.

The first line, known as the header, must end with a colon but the rest does not.

Functions are not used until they are called so if I wanted to call boy, I would have to write something like this:

>>> boy()
'I am indeed a boy'

We could put functions in other functions, so let's say I wanted to repeat the statement, then I could write something like

def repeat_boy():
	boy()
	boy()
	boy()

When I call repeat_boy we get the statement from boy three times since we call boy three times within the function repeat_boy

Scope of Variables

The variables created within a function are local and do not exist outside the scope of the function.

With a function like

def hey_hey(line1, line2):
	s = line1 + line2
	print(s)

The string s only exists within the function hey_hey and if we call it outside of this function, it does not exist.
