Equation Plotting with Python

In this tutorial, I will be going into how to code various equations in Python using Matplotlib.

Libraries

You will need matplotlib and numpy in order to do data visualization with Python. Matplotlib is your basic graphing library while numpy makes it easier to deal with matrices as well as more complex operations with linear algebra or statistics.

import numpy as np
import matplotlib.pyplot as plt

%matplotlib inline

Motivating Example with cos(x)

Let's say we want to plot the trigonometric function cos(x).

This can be as simple as 

x = [-4,-3,-2,-1,0,1,2,3,4]
y = np.cos(x)
plt.plot(x,y)
plt.show()

You put in your x-array and then simply use a numpy function and you get a graph that looks like this:

This might look very jagged because we are inputting discrete inputs, instead we could call a numpy function called np.linspace().

x = np.linspace(-4,4)
y = np.cos(x)
plt.plot(x,y)
plt.show()

This would plot over continuous inputs, giving us

Extra Styles and Decorations

Say we wanted to label the axes or add titles. This is incredibly simple with Python.

To label the x and y axes, we add

plt.xlabel("x")
plt.ylabel("y")

which will get us

Then, to add a title as well as a label on the graph, we add

plt.title("Plot of cos(x)") #Title of the plot
plt.plot(x,y, label = "cos(x)") # Add the label = "cos(x)" so it shows up on the legend
plt.legend() # so the label shows up

Lastly, let's try to add another graph on it and then also change the style.

plt.style.use("ggplot") # Style of the Plot
y2 = np.sin(x)
plt.plot(x,y2, label = "sin(x)") # adding sin(x) to the plot
plt.title("Plot of cos(x) and sin(x)") # changing title

Conclusion

Through this blog, you were able to learn how to plot mathematical formulas like cos(x) quite easily. It goes beyond just this trigonometric function, try for yourself!
