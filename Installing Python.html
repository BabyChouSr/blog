Installing Python

  1. Go on python.org and download it.

2. Run the executable program and make sure to add to PATH

Installing Libraries

After downloading python, press Windows Key + R Then type cmd and press enter.

You Will Need the Following Commands:

  • python -m pip install matplotlib
  • python -m pip install scikit-learn
  • python -m pip install numpy
  • python -m pip install scipy
  • python -m pip install pandas
  • python -m pip install seaborn
  • python -m pip install jupyter notebook

Drag the commands in the command prompt one by one (make sure each is downloaded before moving on)

Installing PyCharm

  1. Go on the jetbrains website to download Pycharm
  1. Open up a new project.
  1. Go To File - > Settings -> Project Interpreter -> Set as Python 3.7(Whatever Version Python You Got)
