Elements in a List

A list is a sequence of values that are called elements.

We create lists by enclosing elements utilize square brackets: [ and ]

We look at the indices of a list just like the indices of a String, starting at 0 for the first item, 1 for the second item, 2 for the third item and so on.

Examples:

x = [10,20,30,40]
y = ['Hey',2,4.0,True]
z = ['A','B','C']

You can also create an empty list that looks something like this 

x = []

Lists are Mutable

Mutable just means that we are able to assign certain values to certain elements in the array.

Example:

Let's say we have an array

x = [1,2,3]

Since lists are mutable, we could then change the second element for example to 5

x[1] = 5
print(x)
>>> [1,5,3]

This assignment statement changes the second element from 2 to 5.

Traversing a list

We can traverse each element within a list using a for loop.

numbers = [1,2,3,4,5]
for num in numbers:
	print(numbers)
>>> 1 
>>> 2 
>>> 3
>>> 4
>>> 5

This loop is good for just reading the elements however, if we want to assign certain values or change those values, we need indices.

Thus, we would use the range function:

for i in range(len(numbers)):
	numbers[i] = numbers[i]*2
print(numbers)
>>> [2,4,6,8,10]

This loop traverses from the 0th index to the index len(numbers) - 1

List operations

We could use the addition operator to concatenate lists.

a = [1,2,3]
b = [4,5,6]
c = a+b
print(c)
>>> [1,2,3,4,5,6]

We could even multiply lists which multiply the number of times that an element occurs.

y = [1,2,3] * 3
print(y)
>>> [1,2,3,1,2,3,1,2,3]

List Slices

We could use the slice operator in order to get a part of the entire list based on a starting and ending index.

This works similar to Strings and substrings except with Strings, each element is a character but for lists, each element is a certain value.

a = [1,2,3,4,5,6]
b = a[1:3]
print(b)
>>> [2,3]

The slicing begins at index one and ends at (3-1) or 2.

We could omit the first index to indicate slicing from the second index all the way to the beginning.

a = [1,2,3,4,5,6]
c = a[:3]
print(c)
>>> [1,2,3]

Conversely, we could omit the second index to indicate slicing from the first index all the way to the end 

a = [1,2,3,4,5,6]
d = a[3:]
print(d)
>>> [4,5,6]

List Methods

Python has several important methods that operate on lists.

append adds a new element to the end of a list

t = ['a','b','c']
t.append('d')
print(t)
>>> ['a','b','c','d']

extend takes another list and appends it to the original list.

t = ['a','b','c']
y = ['d','e','f']
t.extend(y)
print(t)
>>> ['a','b','c','d','e','f']

sort arranges the elements in ascending order.

z = [2,3,6,5,1,4]
z.sort()
print(z)
>>> [1,2,3,4,5,6]

Deleting Elements

pop allows us to delete elements given a certain index

t = ['a', 'b', 'c']
x = t.pop(1)
print(t)
>>> ['a', 'c']

remove lets us delete elements given the element to remove but not the index

t = ['a', 'b', 'c']
t.remove('b')
print(t)
>>> ['a', 'c']
