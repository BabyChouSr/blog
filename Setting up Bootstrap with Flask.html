Posted by Christopher Chou on April 16, 2020

Setting up Bootstrap with Flask

The motivation behind setting up bootstrap with flask is to be able to design your website while linking it to your backend. For me, I can use Flask so that I can upload my model onto my website which is incredibly powerful. Not only that but I can use Flask to create a database for users. It truly allows you to do much more than plain html. However, we still want our website to look good which is why I utilize bootstrap, especially during Hackathons.
