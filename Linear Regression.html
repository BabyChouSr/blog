Linear Regression Theory

Posted by Christopher Chou on April 15, 2020

Linear Regression

The job of linear regression is to create a linear model that best fits data points (x,y). Using linear regression, we try to create a linear model that maps from independent variable (x) to dependent variable (y) while minimizing the loss.

This can be visualized with this graph

Model

We try to find the best slope and intercept that best fits the data points.

However, in order to declare what is the best model, we have to see how far away our current model is from the actual values. To do this, we have to declare the loss function.

Cost Function

First, we have to define our loss function or how far away our observed value is from our predicted value. This is done by taking the residual (difference between the actual and the predicted) but we square it because negative and positive should have the same impact.

The Cost Function can be defined as the average of the sum of the residuals which is known as the Mean Squared Error:

Our job with linear regression is to minimize this mean squared error.

Gradient Descent
