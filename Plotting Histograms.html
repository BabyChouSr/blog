Plotting Histograms

Posted by Christopher Chou on April 26, 2020

Plotting Histograms

In this blog, I go over how to plot histograms using matplotlib which allows us to see how variables are distributed. This is incredibly important in statistics because it allows us to make inferences based on the distribution of the data.

I have also created a video on this topic:

Importing Libraries

import matplotlib.pyplot as plt
import numpy as np

We import our libraries that we need through these two lines

Introduction to Histograms

We are going to start plotting histograms with just purely random data which is spawned by utilizing the random function from numpy.

I am going to use the style of ggplot to make it look a little more stylish, but then again this is just dummy data.

X is just a numpy array of size 1000 ranging from 0 to 10.

I created the bins by using np.linspace which helps to create 100 partitions from 0 to 10. This would create an array that looks something like: 

plt.style.use('ggplot')
x = np.random.rand(1000) * 10
bins = np.linspace(0,10,num = 100)
plt.hist(x, bins = bins, edgecolor = 'black',color = "#a8329e", label = "hey dummy")
plt.title("Dummy Data isn't Smart")
plt.xlabel("Ages of Dummy")
plt.ylabel("Frequency of Dummy")
plt.legend()

Plotting Normal Distribution

A normal distribution is essential in statistics for any statistical test such as confidence intervals and hypothesis testing.

xnorm is just the array of size 1000 with mean 0 and standard deviation 1. I put it at 0 and 1 because that's the definition of a standardized normal distribution.

I defined the bins from -3 to 3 because by the empirical rule which is (68,95,99.7), we would expect the entire distribution to be within -3 to 3.

From this, we get to see a cool looking normal distribution which we might be able to see if a certain test is applicable.

xnorm = np.random.normal(0, 1, 1000)
bin_norm = np.linspace(-3,3,num=100)
plt.hist(xnorm, bins = bin_norm, edgecolor = 'black',color = "#42c8f5", label = "hey dummy2")
plt.title("Dummy Data isn't Smart")
plt.xlabel("Ages of Dummy")
plt.ylabel("Frequency of Dummy")
plt.legend()

Plotting Vertical Lines

What if we wanted to signify a certain feature? To do so, we would use a function called plt.axvline()

xnorm = np.random.normal(0, 1, 1000)
bin_norm = np.linspace(-3,3,num=100)
plt.hist(xnorm, bins = bin_norm, edgecolor = 'black',color = "#45de73", label = 'hey dummy3')
median = 0
plt.axvline(median, color = 'red', label = "Median")
plt.title("Dummy Data isn't Smart")
plt.xlabel("Ages of Dummy")
plt.ylabel("Frequency of Dummy")
plt.legend()

This is the same function as before except we added a certain function called plt.axvline(). We can assume that the median and mean are at 0 by definition of a normal distribution. Anyways, the color is red and we are able to label it which will show up on the legend.

Conclusion

There we go! That's really it with histograms. All we have to do is:

  1. Find an x-array with frequency
  1. Partition the plot into bins
  1. The rest is just styles!
