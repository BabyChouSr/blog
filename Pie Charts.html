Pie Charts

In this blog, I will be going over how to plot pie charts which is useful to see the relative area between certain data. We rely on the user's intuition of area in order to create a meaningful graph.

Importing Libraries

We first import the libraries that we need

import numpy as np  
import matplotlib.pyplot as plt
%matplotlib inline

Background Info

Pie Charts work by utilizing the proportions of the numerical data.

In this case, with slices = [10,20,30,40], the plt.pie() function will calculate the proportion of each of those parts over the total.

Mathematically it would look something like this:

The labels are the name of each of the slices of the graph. The color is the color that we give our pies. 

slices = [10,20,30,40]
labels = ['TEN','TWENTY','THIRTY','FORTY']
colors = ['blue','green','yellow','purple']

plt.style.use('dark_background')
plt.pie(slices, labels = labels, colors = colors)
plt.title("pies are good")
plt.tight_layout()

Here is our pie chart. However what if we wanted to emphasize certain features on the graph? Everyone is just glued together right now and we don't seem to get a clear message from all this.

Exploding

We can "explode" part of the graph out of the original graph to create contrast and to demonstrate the differences in the graph

explode = [0,0,0.2,0]
slices  = [10,20,30,40]
labels = ['TEN','TWENTY','THIRTY','FORTY']
colors = ['blue','green','yellow','purple']

plt.style.use('dark_background')
plt.pie(slices, labels = labels, explode = explode, colors = colors, autopct = '%1.2f%%')
plt.title("pies are good")
plt.tight_layout()

Notice how I added an "explode" array to tell matplotlib how much I want to explode it by. 0 just means that we won't explode it at all while the 0.2 tells matplotlib to explode a little. The autopct parameter tells matplotlib to show the percentages up to 2 decimals.

Conclusion

It's really THAT easy to plot pie charts using matplotlib. A word of caution is that when there are multiple groups (I'd say over 5), pie charts are rarely a good choice. All the data gets smushed together and is hard to see. Thus, I would be very careful as to whether or not a pie chart really conveys the message that you want.
