Setting Up Flask

Posted by Christopher Chou on April 15, 2020

Why?

What is a virtual environment? Think about it like a portable environment. It’s like having a mini-system interpreter in your own environment. Its disposable and portable which is why its useful to deploy on your own server.

Install virtualenv

Creating a Virtual Environment

cd desktop - changes to the desktop directory because that’s where I want my virtual environment

virtualenv “NAME OF VIRTUAL ENVIRONMENT” (I named it environment)

Running a Virtual Environment

cd environment - changes to the environment directory

scriptsessentially activates the virtual environment

(environment) on the side just shows that the virtual environment is running

Creating folder for flask project

mkdir (make directory or folder)

Go within your folder and make a new python project

Setting the System Interpreter for your Project

Install Flask

Create A Python File

call it app.py

The following program is just the “hello world” of flask

from flask import Flask, escape, request

app = Flask(__name__)

@app.route('/')
def hello():
    name = request.args.get("name", "World")
    return f'Hello, {escape(name)}!'

Start the App
