Setting up Particle.js

Particle.js is an incredibly cool looking background add-on that makes your web application look extremely progressive and nice.

Just see for yourself!

Downloading from Github

First, you need to download two javascript files from Github. The link is down below

Press the green "clone or download" button to download the entire respository.

The two files you will be looking for after downloading it is "particles.js" and "app.js."

Particle.js is simple enough to find as it is in the first folder you see.

To find app.js, you have to go into demo > js > app.js

You want to extract these two files into your main folder that you are working in. This should look something like this:

I created my own html file and a css file.

Some Quick Coding

In the html file, you should call the two particles script and the app script as well as create a div so that the particles can show up. 

<html>
    <head>
        <link rel = "stylesheet" href = "style.css">
    </head>
    <body>
        <div id = particles-js></div>
        <script src = "particles.js"></script>
        <script src = "app.js"></script>
    </body>
</html>

I also called the style.css file that I am using

The css file should look something like this

body {
    margin: 0;
    padding: 0;
    height: 100vh;
}

#particles-js {
    height: 100%;
    background: #6666ff;
}

The background color is whatever you want, this one just allowed my background to look blue which I like.

And That's really IT!
